## 🚀 ROADMAP ПО ПРИОРИТЕТАМ (на основе ChatGPT review)

> Главный фокус: Усиливать core loop, а не расширять функции (никакой крипты, веб-дашбордов и сложной аналитики). Бот должен быть "антистрессом для денег", а не "хранилищем цифр".

### PRIORITY 1: Proactive AI (Nudges)
> **Goal:** Превратить инструмент в привычку. Бот должен вести пользователя.
- [ ] **Предупреждения:** `⚠️ На продукты осталось 6к до конца месяца.`
- [ ] **Positive reinforcement:** `🔥 Ты уже 2 недели не выходишь за бюджет.` или `Ты сократил долг на 66к за месяц — это очень сильный прогресс.`
- [ ] **Возврат (Re-engagement):** `Ты давно не обновлял расходы. Продолжим вести бюджет?`
- **Technical Implementation:** Настроить `apscheduler` или cron-задачу, которая раз в день/неделю сканирует БД, собирает `envelopes`, скармливает их LLM с системным промптом "сгенерируй персонализированный пуш" и отправляет пользователю.

### PRIORITY 2: WOW-moment после настройки (Forecasts)
> **Goal:** Дать ощущение "у меня впервые появился понятный план".
- [x] **Прогноз будущего в Дашборде:** Бот должен считать месяцы до закрытия долгов и выводить это текстом.
- [x] *Пример:* `🔥 Хорошая новость: Если придерживаться плана, кредитка закроется примерно за 1 месяц, долг брату — за ~2 месяца. После этого сможем откладывать ~20к/мес на ноутбук.`
- **Technical Implementation:** Дописать логику `get_financial_insight` в `transactions.py` для генерации прогнозного блока (считать скорость гашения на основе `monthly_income` / `free_cash`).

### PRIORITY 3: Voice UX [x]
> **Goal:** Ультра-быстрый парсинг голоса.
- [x] Человек должен иметь возможность наговорить: `Брат вернул 5к, а еще купил хавки на 2300`.
- [x] Бот должен всё понять, распределить и не задавать лишних вопросов.
- **Technical Implementation:** Интеграция Telegram Voice (Ogg/Opus) -> Whisper API -> Текст -> Существующий LLM пайплайн.

### PRIORITY 4: Character / Personality
> **Goal:** Сделать бота "спокойным, уверенным финансовым навигатором" без кринжа.
- [ ] Обновить `prompt_vibe` пользователя или жестко зашить в систему тональность (спокойный, supportive, pragmatic).

### PRIORITY 5: Найти killer audience
- Тестирование 3 сегментов:
  1. Люди с долгами (emotional pain)
  2. Фрилансеры (chaos pain)
  3. Люди "деньги исчезают" (массовый сегмент)
